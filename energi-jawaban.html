Soal Energi

Kunci Jawaban

  1. Diketahui :
    \(m = 200\) gram = \(0,2 kg\)
    \(h = 4,05\) m (jatuh bebas)

    Ditanya : \(Ek = ?\)
    Jawaban :
    $$Ek = \frac12 mv^2$$ Kecepatan jatuh bebas dihitung sebagai berikut. $$v=\sqrt{2gh}$$ $$v = \sqrt{2\times10\times4,05} = \sqrt{81} = 9\text{ m/s}$$ Energi kinetik dapat dihitung sebagai berikut. $$Ek = \frac12\times0,2\times9^2 = 8,1\text{ joule}$$
  2. Diketahui :
    \(v_1 = 2\) m/s (ada dua keadaan)
    \(Ek_1 = 2,5\) joule
    \(v_2 = 6\) m/s (massanya tidak disebutkan = dianggap sama)

    Ditanya : \(Ek_2 = ?\)
    Jawaban : $$Ek = \frac12 mv^2$$ karena ada dua keadaan, maka kita lakukan perbandingan. $$\frac{Ek_1}{Ek_2} = \frac mm\times\frac{v_1^2}{v_2^2}=\frac{1}{1}\times\left(\frac{2}{6}\right)^2 = \frac19$$ lakukan perkalian silang akan diperoleh \(Ek_2 = 22,5 \text{ joule}\).
  3. Diketahui :
    \(V = 200\) liter
    \(h = 4,0\) m
    \(g = 10\) m/s²

    Ditanya : \(Ep = ?\)
    Jawaban : $$Ep = mgh$$ massa diperoleh dari volume aair dengan massa jenis air \(\rho = 1\) kg/liter, maka \(m = 1\) kg
    $$Ep = m\cdot g\cdot h = 200\cdot10\cdot4 = 8000 \text{ joule}$$
